Demo1 - Copy Files from A.C to Required Nodes
==============
YML File :  myplaybook1.yml
------------------------------------
- hosts: all
  tasks:
    - name: Copy Files
      copy: 
        src: hello.txt
        dest: /home/ubuntu

=========================================================
scp myplaybook1.yml ubuntu@18.116.64.4:/home/ubuntu/myansible
=========================================================

ansible all -a "ls -l /home/ubuntu"

ansible-playbook myplaybook1.yml 

ansible all -a "ls -l /home/ubuntu"

ansible-playbook myplaybook1.yml 

ansible all -a "ls -l /home/ubuntu"

Explanation :
1) hosts :  where you want to perform the Operation

all/myjlcweb/myjlcdb/IP/localhost

2) tasks : array of tasks to perform
 
3) name : task name

4) copy  : module name

Demo4 - Copy File and Delete File
===================

------------------------------------

  become: yes
  gather_facts: yes
  tasks:
    - name: Create File
      file: 
        path: /home/ubuntu/appCode/Test.java

Demo9 - Loops using loop
===================
YML File :  myplaybook9.yml
------------------------------------
- hosts: all
  become: yes
  gather_facts: NO
  tasks:
    - name: Creating {{item}}
      file: 
        path: "/home/ubuntu/appCode/{{item}}"
        state: touch

        - test1.txt
        - test2.txt
        - test3.txt
        - test4.txt
        - test5.txt
